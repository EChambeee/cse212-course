CSE 212 | Programming with Data Structures

07 Teach: Group Collaboration

Overview

With your assigned study group, you should spend up to 1 hour collaborating together on the 07-Prove problems. You must not share your solutions with each other. During this time together, you should share strategy, technique, and examples to help each other learn the material. Make sure that each person in your group has an oppourtunity to share ideas and ask questions. You should prepare yourself for this activity by starting or completing several of the problems from the individual Prove assignment.

As you solve the problems, remember to use the principles learned in the first 6 lessons of the course:

  • Problem Solving Strategy: Understand the problem, Plan & Design the Solution, Implement the Solution, and Evalute the Solution

  • Evaluate Performance of Alternative Solutions: Use Big-O to determine the best solution

  • Understanding Code using Reviews: Analyze any code you have been given

  • Finding Defects using Tests: Write tests to determine if your solution is working

  • Answering Technical Interview Questions: Imagine that you were asked one of these questions during an interview. Remember to ensure that you fully undestand the problem and work through different scenarios manually. Consider how the data structure you are using will help you solve the problem. A notebook or a whiteboard can be very useful in this process.

CSE 212 | Programming with Data Structures

04 Prove: Individual Assignment

Overview

This assignment must be completed individually. You should not solve these problems with other students. If you obtain help from a tutor, the tutor should help understand principles but should not help you solve these problems.

In this assignment you will be asked to use testing to find errors in the different implementations of the Queue data structure. You must use the template provided below in the instructions to complete this assignment.

Instructions

  1. Download the following template: cs241_prove1.doc

  2. Download the following code file: priority_queue.py

  3. Note that the requirements for the Priority Queue are written at the top of the code file. Execute the test cases provided for you in the Priority Queue code. Using the template, summarize the results of all the tests.

  4. Based on the failed test cases, fix the Priority Queue code. Using the template, summarize all the fixes that yoou made to the code.

  5. Download the following code file: circular_queue.py

  6. Based on the requirements for the Circular Queue written at the top of the code file, write test cases that over the requirements. Each test case should include a Scenario Description and an Expected Result. Document tese tests cases in the template.

Submission

When complete, upload your assignment to I-Learn.

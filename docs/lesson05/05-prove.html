Instructions

There are 2 parts to this assignment.

Part 1 - Set Operations

  1. Download the following code file: set_operations.py

  2. Assume you have been asked in an interview to describe how you wrould write a function to find the intersection of 2 sets and a function to find the union of 2 sets. An intersection will return a set containing items that are common between the 2 sets. A union will return a set containing all items from both sets. There is an intersection (also the & operator) and union function (also the | operator) already in Python. For this assignment, do not use these capabilities already in Python.

  3. In response, first write down an answer to the question as if you were giving a verbal response with an emphasis on the purposes, behavior, and/or performance of a Set. Your response should be no more than a 30 second answer if spoken out loud.

  4. Second, implement both functions in the code file provided above.

  5. You will provide your resposne and code in the I-Learn quiz.

Part 2 -

  1. Download the following code file: largest_unique_substring.py

  2. Assume you have been asked in an interview to describe how you can find the length of the longest unique substring within a text input. For example, if the text is "ABCABCDEABABCD" it would return 5 because "ABCDE" is the longest unique substring. Using a Set you should be able to write code that runs in O(n) time.

  3. In response, first write down an answer to the question as if you were giving a verbal response with an emphasis on the purpose, behavior, and/or performance of a Set. Your response should be no more than a 30 second answer if spoken out loud.

  4. Second, implement the function in the code file provided above.

CSE 212 | Programming with Data Structures

05 Teach: Group Practice

Overview

You should complete these practice problems with one or more members from your assigned study group. You will be solving problems using sets by first discussing how the behavior, purpose, and/or performance of the set can help you arrive at the answer (ref: Syllabus Course Outcome 5). When asked to solve a problem in an interview, the interviewer will be very interested in your knowledge of data structures even if you can't fully solve the entire problem verbally. This practice is similar to the individual work you will complete in 05-Prove. This practice should take no more than one hour. If you are about to run out of time, you may look at the solutions and discuss them.

Instructions

Problem 1 - Unique Letters

  1. Download the following code and load it into your code editor: unique_letters.py

  2. This code will determine if a string has all unique letters. For example, "abcdefg" has unique letters but "abccdefg" does not because the "c" is repeated more than once. The code is implemented using loops which results in an O(n^2) performance.

  3. Answer this question together: how can the unique letter function be written with better performance using a set data structure? As you answer this question, talk about the behavior, purpose, and/or performance of a set to help you arrive at the answer.

  4. When you think you know how to solve the problem (take no more than five minutes), read the answer to the question and compare with your initial answer.

  5. Based on your understanding, modify the unique_letters function so it uses a set instead.

  6. When you are done, take a look at the solution and discuss it together.

Problem 2 - Display Sums

  1. Download the following code and load it into your code editor: display_sums.py

  2. Your goal is to write a program that will find and display all pairs of numbers in a list that sum up to 10. This could be done in O(n^2) with a loop within a loop. However, using a set, this can be done in O(n) time. You should assume that list of numbers provided has no duplicates.

  3. Answer this question together: how can you solve this problem using a set data structure? As you answer this question, talk about the behavior, purpose, and/or performance of a set to help you arrive at the answer.

  4. When you think you know how to solve the problem (take no more than five minutes), read the answer to the question and compare with your initial answer.

  5. Based on your understanding, modify the unique_letters function so it uses a set instead.

  6. When you are done, take a look at the solution and discuss it together.

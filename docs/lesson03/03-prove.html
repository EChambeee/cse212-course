CSE 212 | Programming with Data Structures

03 Prove: Individual Assignment

Overview

This assignment must be completed individually. You should not solve these problems with other students. If you obtain help from a tutor, the tutor should help you understand principles but should not help you solve these problems. It is an honor code violation to seek the solution to these exact problems from others including the internet (including sites that allow students to share their solutions).

You will summarize all your answers in the associated I-Learn quiz. Please note that you only get one submission attempt with the I-Learn quiz.

Instructions

There are 3 parts to this assignment.

Part 1 - First Mystery Stack

  1. Open the following PDF copy of code: mystery_stack_1.pdf

  2. Using techniques learned in the reading and practice this week, determine what this code does.

  3. What is the output of the function if the input text is equal to the following:

    • racecar

    • stressed

    • a nut for a jar of tuna

  4. You will summarize your findings in the I-Learn quiz.

Part 2 - Second Mystery Stack

  1. Open the following PDF copy of code: mystery_stack_2.pdf

  2. Using techniques learned in the reading and practice this week, determine what this code does.

  3. Propose a value for the input parameter text that will result in the function doing the following:

    • Returning an answer of positive 5

    • The display of "Invalid Case 1!"

    • The display of "Invalid Case 2!"

    • The display of "Invalid Case 3!"

    • The display of "Invalid Case 4!"

  4. You will summarize your findings in the I-Learn quiz.

Part 3 - Reflection

The last question in the I-Learn quiz will ask you to write a paragraph comparing the reading of code to the reading of a book. You should also consider how you can improve your reading of code in the next year. You should use your experiences from the Group Practice and this Assignment to help formulate your response.

Part 1 - List Comprehension

  1. Open up the following code: 01-prove_multiples_of.py

  2. Implement the multiples_of function. The function should create and return a list of multiples of a number. The starting number and the number of multiples are provided as inputs to the function. For example, multiples_of(3,5), where the 3 is the starting number and 5 is the number of multiples, would result in [3, 6, 9, 12, 15]. You must implement this using a single list comprehension. You may want to write it first without a list comprehension and then modify it.

Part 2 - Solving a Complicated Problem

  1. Open up the following code: 01-prove_combined_sorted_lsits.py

  2. Come up with a plan on how to implement the combine_sorted_lists function. This function receives two sorted lists and will return a new sorted list containing the values from both input lists. For example, if list 1 is [1, 3, 7] and list 2 is [2, 6, 10, 11], then the resulting list would be [1, 2, 3, 6, 7, 10, 11]. You cannot use the min, max, sort, or sorted Python functions to solve this problem. Remember the suggestion to "solve the problem manually". As you create the sorted list, how do you know that you should pick the 1 before the 2 and then pick the 2 before the 3. Write down a process for doing step by step and then translate this to code.

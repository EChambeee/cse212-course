CSE 212 | Programming with Data Structures

14 Prepare: Course Conclusion

Overview

Congratulations! You have completed CSE212. In this course you learned many important software engineering skills that will be useful to you in your career. These skills have included the following:

  • Problem Solver: Apply a systematic approach to writing code to solve problems.

  • Evaluator: Evaluate the performance of alternative solutions using big O notation.

  • Reviewer: Predict the purpose and behavior of existing code.

  • Tester: Discover defects using testing in code.

  • Conversant: Articulate the answers to technical questions involving data structures.

  • Self-Reliant: Solve problems independently.

In addition to learning these software engineering skills, you have also learned about numerous data structures. You have explored the purpose, performance, and application of each of the data structures. These data structures have included the following:

  • Dynamic Arrays

  • Stacks

  • Queues

  • Sets

  • Maps

  • Linked Lists

  • Trees

All of these topics are likely to be tested in your future job and internship interviews. Regular review and use of these skills and data structures should be part of your future plans.

The final project is due on the last day of the semester. You should submit your final project in 14-Prove. There is no final exam.
